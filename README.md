# Borne d'arcade
Projet réalisé par Aletrut Thibaut, Gadefait Etienne et Horion-Coudène Claire.

Le but est de créer une borne d'arcade à partir d'une carte ProMicro, de boutons, et d'un écran d'ordinateur. 
Tout ceci réalisé avec des composants Arduino.
Le projet principal est donc lancé avec l'IDE Arduino avec le fichier Borne-arcade.ino.
Pour faire fonctionner le projet, il est nécessaire au préalable de télécharger la librairie ArduinoSTL, de l'ajouter dans votre propre dossier Arduino/libraries puis de supprimer le fichier "new_handler.cpp".

Nous avons utilisé un joystick pour certains tests. Finalement, nous utilisons une carte Pro Micro avec quatre boutons pour les déplacements et un bouton central de sélection. L'affichage se fait au travers de la connexion Serial avec le pc.

Un diagramme de classe est fourni avec le projet.
La rapport apporte plus de précsions sur la conception du projet.
