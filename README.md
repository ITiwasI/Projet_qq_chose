# Borne d'arcade
Projet réalisé par Gadefait Etienne, Horion-Coudene Claire et Aletrut Thibaut.

Le but est de créer une borne d'arcade à partir d'une esp, de boutons, et d'un écran d'ordinateur. 
Tout ceci réalisé avec des composant Arduino.
Le projet principale est donc lancé avec l'IDE Arduino avec le fichier Borne-arcde.ino.

Nous avons utilisez un joystick pour certain test.
et l'affichage se fait au travers de la connexion Serial avec le pc.

Un diagramme de classe est fournit avec le projet.
